# Theme Viewer — alternate font candidates

Free fonts staged for `tools/theme-viewer.html` font-pair cards.
Shipped theme defaults live under `themes/<name>/fonts/`; these files are
**alternates** you can try in the viewer and optionally promote into a theme.

All are free to redistribute with PxD (keep the matching OFL/Apache license text).
